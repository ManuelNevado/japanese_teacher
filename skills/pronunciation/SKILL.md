# Skill: Japanese Pronunciation Guide 発音ガイド

## Descripción
Guía completa de pronunciación japonesa adaptada especialmente para hispanohablantes, cubriendo desde el sistema de mora hasta el pitch accent avanzado.

---

## El Sistema de Mora (モーラ)

El japonés no usa sílabas como el español — usa **moras** (拍, haku). Cada mora tiene exactamente la misma duración.

### Diferencias con el Español
```
Spanish:  "sí-la-ba" (sílabas con peso variable)
Japonés:  "mo-ra" (cada unidad = exactamente una mora)
```

### Tipos de Mora
| Tipo | Ejemplo | Moras |
|------|---------|-------|
| Vocal simple | あ (a) | 1 |
| Consonante + vocal | か (ka) | 1 |
| Vocal larga | ああ (aa) | 2 |
| ん (n) silábica | ほん (ho-n) | 2 |
| っ (oclusiva) | きって (ki-t-te) | 3 |

## Las 46 Hiragana — Pronunciación Base

### Fila A (あ行)
| Kana | Romanización | Como en español |
|------|-------------|----------------|
| あ | a | como "a" de "casa" |
| い | i | como "i" de "sí" |
| う | u | entre "u" e "ü" — labios sin redondear |
| え | e | como "e" de "mesa" |
| お | o | como "o" de "rosa" |

> ⚠️ El **う** es el más difícil para hispanohablantes. No redondees los labios.

### Sonidos Inexistentes en Español
| Kana | Descripción | Truco |
|------|-------------|-------|
| ふ (fu) | Fricativa bilabial, no "fu" inglés | Sopla suave sin morder el labio |
| ら/り/る/れ/ろ | Lateral/flap, no "r" española | Como la "r" suave de "pero" en español |
| つ (tsu) | "ts" + "u" | Como "pizza" seguido de "u" |
| し (shi) | No es "si" | Como "shi" inglés |
| ち (chi) | No es "ti" | Como "chi" de "China" |
| じ/ぢ (ji) | No es "yi" | Como "j" suave del inglés |
| ず/づ (zu) | No es "su" | Como "z" inglesa |

## Pitch Accent (声調) — Acento Tonal

El japonés es un idioma de **acento de altura** (*pitch accent*), no de intensidad como el español.

### Los 4 Patrones de Acento (Japonés Estándar de Tokyo)

#### Patrón Heiban (平板型) — Sin bajada después del acento
```
Todas las moras después de la primera suben y se mantienen altas.
Ejemplo: はし (hashi = palillos)
mora 1: HA (baja)
mora 2: SHI (alta) → se mantiene alta
```

#### Patrón Atamadaka (頭高型) — Acento en la primera mora
```
La primera mora es alta, el resto baja.
Ejemplo: はし (hashi = borde)
mora 1: HA (alta)
mora 2: SHI (baja)
```

#### Patrón Nakadaka (中高型) — Acento en el medio
```
Sube después de la primera, baja después de cierta mora.
Ejemplo: たまご (tamago = huevo)
mora 1: TA (baja)
mora 2: MA (alta)
mora 3: GO (alta → baja)
```

#### Patrón Odaka (尾高型) — Acento en la última mora
```
Los más difíciles de notar.
mora 1: baja
mora 2..N: altas
pero si sigue partícula: baja
```

### Pares Mínimos con Pitch Accent
```
はし (borde)   → HA↑SHI↓  — Atamadaka
はし (palillos) → HA↓SHI↑  — Heiban
はし (puente)  → HA↓SHI↑  — Heiban (contexto diferencia)

あめ (lluvia)  → AME↑  — Atamadaka
あめ (caramelo) → A↓ME↑ — Heiban
```

## Consonantes Geminadas (っ)

La **っ** (sokuon) dobla la consonante siguiente y crea una pausa de 1 mora.

```
きって (ki-t-te) = ki + pausa + te  → "ki...te" (estampilla)
きて  (ki-te)   = ki + te           → "kite" (ven)
```

Truco: Piensa en la pausa como un golpe de tambor silencioso.

## Problemas Frecuentes de Hispanohablantes

| Error Común | Corrección |
|-------------|-----------|
| Pronunciar "r" japonesa como "r" española vibrante | Usar solo una toque pequeño de la lengua (flap) |
| Redondear labios en "u" | Mantener labios extendidos/neutrales |
| Ignorar vocales largas (oo, uu) | Duplicar exactamente la duración |
| Ignorar consonantes geminadas (っ) | Pausar una mora completa |
| Pronunciar ん como "n" dental | La ん se asimila a la consonante siguiente |
| Añadir vocal a consonantes finales | El japonés no tiene consonantes finales (salvo ん) |

## Dialectos

| Dialecto | Región | Diferencias Clave |
|----------|--------|------------------|
| 標準語 Hyōjungo | Tokyo (estándar) | Base del japonés enseñado |
| 関西弁 Kansai-ben | Osaka, Kyoto | Pitch accent invertido frecuente, ちゃう=違う |
| 東北弁 Tōhoku-ben | Norte de Honshu | Vocales ensordecidas, difícil de entender |
| 沖縄弁 Okinawa-ben | Okinawa | Casi una lengua aparte |

## Notas para el Agente

- Para N5, enfócate en pronunciación correcta de hiragana/katakana básicos
- El pitch accent puede introducirse gradualmente — no abrumar al principio
- Siempre que corrijas pronunciación, muestra la versión correcta con descripción articulatoria
- El japonés de anime tiene acentos exagerados o ficticios — advertir al estudiante
