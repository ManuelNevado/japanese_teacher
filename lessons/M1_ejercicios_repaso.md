1. ねこはつくえのしたがいます。→

4. まいあさにコーヒーをのみます。→

4. つくえのしたにかばんが___。

### 5.2 Transforma a negativo

1. へやにいぬがいます。→
2. つくえのうえにほんがあります。→
3. がっこうにともだちがいます。→
4. テーブルのしたにかばんがあります。→

## BLOQUE 11 — Comprensión lectora

### Texto 1

> わたしのなまえはたろうです。がくせいです。まいあさしちじにおきて、はちじにがっこうへいきます。がっこうにはともだちがたくさんいます。わたしのへやにはつくえとほんだなとテレビがあります。つくえのうえにほんとノートがあります。いすはつくえのまえにあります。

1. ¿Cómo se llama?
2. ¿A qué hora se levanta?
3. ¿A qué hora va a la escuela?
4. ¿Qué hay en su habitación?
5. ¿Dónde está la silla?

---

### Texto 2

> わたしのかぞくはよにんです。ちちとははとあにとわたしです。ちちはいしゃです。はははせんせいです。あにはだいがくせいです。わたしはこうこうせいです。うちはにぎやかです。まいばんかぞくでごはんをたべます。

1. ¿Cuántas personas hay en la familia?
2. ¿Qué hace la madre?
3. ¿Hay alguna hermana?
4. ¿Cómo es la casa?
5. ¿Qué hacen todas las noches juntos?

## BLOQUE 12 — Ejercicios mixtos

### 12.1 Verdadero o falso — corrige los falsos

1. "El gato está encima de la mesa" → ねこがつくえのしたにいます。
2. "Esta es mi bolsa" → これはわたしのかばんです。
3. "La habitación no es pequeña" → へやはちいさいです。
4. "¿Hay un libro?" → ほんがいますか？
5. "Estudio todos los días a las 9" → まいにちくじにべんきょうします。

---

### 12.2 Ordena las palabras

1. います / に / ねこ / の / つくえ / した
2. です / の / わたし / かばん / これ / は
3. あります / の / テレビ / に / へや / なか
4. じゅうにじ / まいにち / べんきょうします / に
5. きれい / しずか / は / で / このへや / です

---
