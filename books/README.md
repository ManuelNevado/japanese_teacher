## Módulo 1 — Temas cubiertos

- **Cap. 1**: Hiragana — lectura, escritura, caracteres confusos, っ y pequeños
- **Cap. 2**: Estructura XはYです, negación じゃないです, preguntas con か
- **Cap. 3**: Vocabulario base — pronombres, demostrativos, objetos, personas
- **Cap. 4**: Adjetivos い y な — uso como predicado y modificador
- **Cap. 5**: Existencia (います/あります), partícula に para ubicación, partícula の
- **Cap. 6**: Números 1–10.000, contador ～つ, contadores ～まい/～ほん/～ひき, kanji numéricos

## Cómo usar

1. Lee el capítulo del **libro de texto** para entender el concepto.
2. Haz los ejercicios del **libro de ejercicios** sin mirar las respuestas.
3. Comprueba tus respuestas al final de cada sección.
4. El **Examen Final** de cada módulo evalúa todo lo aprendido.

## Nivel

JLPT N5 — nivel inicial. Cada módulo cubre ~4-8 lecciones del currículum completo N5→N1.
