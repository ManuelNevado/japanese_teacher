## EJERCICIOS CAPÍTULO 2 — Estructura Básica

### Ejercicio 2.1 — Construcción de oraciones (XはYです)

Construye oraciones afirmativas siguiendo el modelo:
> わたし / がくせい → **わたしはがくせいです。**

2. あれ / いぬ → ___________
3. あなた / せんせい → ___________

### Ejercicio 2.2 — Negación (じゃないです)

Niega estas oraciones:

1. これはほんです。→ ___________
2. わたしはせんせいです。→ ___________
3. あれはねこです。→ ___________

### Ejercicio 2.3 — Preguntas (ですか)

Convierte en pregunta y escribe una respuesta negativa:

1. これはかばんです。→ Pregunta: ___________ / Respuesta negativa: ___________
2. あなたはがくせいです。→ Pregunta: ___________ / Respuesta negativa: ___________

### Ejercicio 2.4 — Traducción al japonés

2. Aquello no es un perro. → ___________

4. Sí, soy estudiante. → ___________

### Ejercicio 2.5 — Identificar el error

Cada oración tiene un error. Corrígelo:

1. わたしわがくせいです。→ ___________
2. これはほんじゃないか。→ ___________
3. あなたはせんせいです？→ ___________

### Respuestas — Capítulo 2

**2.1**
1. これはほんです。
2. あれはいぬです。
3. あなたはせんせいです。
4. そのかばんはあたらしいです。

**2.2**
1. これはほんじゃないです。
2. わたしはせんせいじゃないです。
3. あれはねこじゃないです。

**2.3**
1. これはかばんですか。 / いいえ、かばんじゃないです。
2. あなたはがくせいですか。 / いいえ、がくせいじゃないです。

**2.4**
1. これはほんです。
2. あれはいぬじゃないです。
3. がくせいですか。
4. はい、がくせいです。
5. いいえ、せんせいじゃないです。

**2.5**
1. わたし**は**がくせいです。(は no わ como partícula)
2. これはほんじゃない**です**か。(necesita です antes de か)
3. あなたはせんせいです**か**。(か, sin signo de interrogación ?)

## EJERCICIOS CAPÍTULO 5 — Existencia y Ubicación

### Ejercicio 5.1 — います vs あります

Elige el verbo correcto:

1. へやにねこが ___________ 。(います / あります)
2. つくえのうえにほんが ___________ 。(います / あります)
3. こうえんにこどもが ___________ 。(います / あります)
4. かばんのなかにみずが ___________ 。(います / あります)
5. にわにいぬが ___________ 。(います / あります)

### Ejercicio 5.2 — Construcción con に

Construye oraciones de existencia/ubicación:
> ねこ / つくえのうえ → **つくえのうえにねこがいます。**

1. ほん / かばんのなか → ___________
2. いす / つくえのまえ → ___________
3. いぬ / へやのそと → ___________
4. みず / つくえのした → ___________

### Ejercicio 5.3 — La partícula の

Une los elementos con の:

1. わたし + かばん → ___________
2. せんせい + ほん → ___________
3. つくえ + うえ → ___________
4. へや + なか → ___________

### Ejercicio 5.4 — Describir una imagen

Imagina esta escena y escribe 5 oraciones en japonés:

*Hay una habitación. Un gato está encima del escritorio. Un bolso está debajo de la silla. Hay un libro nuevo dentro del bolso. El gato es pequeño.*

1. ___________
2. ___________
3. ___________
4. ___________
5. ___________

2. あります
